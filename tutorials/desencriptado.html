Para desencriptar un texto, se debe ingresar un texto que anteriormente fue encriptado en el text-area, y dar clic en el boton "Desencriptar". Al presionar este boton se verifica que no este vacio el text-area, si no es asi se convierte en un arreglo de caracteres. Luego mediante un arreglo que contenga las vocales, se recorrera el arreglo de caracteres ingresado.

Como se muestra, se hacen dos ciclos, uno que recorre el arreglo de vocales, y otro que recorre el arreglo de caracteres ingresado. Si el arreglo de caracteres ingresado incluye alguna de las combinaciones de letras del encriptado, entonces se reemplaza la vocal por la letra correspondiente. Luego se muestra el texto desencriptado en el div "textFound".
